# AI味模板检测规则

> 不再按"禁用词黑名单"执行。改为**模板叠加检测**：单个信号词在语境中正常使用不禁止；两个以上信号词在同一句中机械叠加 = AI 模板 = 必须改写。

---

## 决策路由

| 你在做什么 | 查阅哪个模块 |
|-----------|-------------|
| 检查单段/单句是否有AI味 | 模板叠加检测 -> 叠加等级 |
| 判断某个信号词该不该删 | 语境判断：有功能 vs 无功能 |
| 扫描全章AI味密度 | 全章扫描策略 |
| 改写AI味重的句子 | 替换策略速查 |

---

## 核心原则

**单用不禁，叠加必杀。**

"缓缓"单独用在动作节奏上（"他缓缓转头"）→ 有功能，保留。
"他缓缓站起身，眼中闪过一丝坚定" → 2个信号词叠加 + 模板句式 = AI模板，改写。

自然文本（参考《每死一次，就多一件宝贝》55万字，评分7.8）：使用"缓缓"126次、"一丝"145次、"轻轻"72次，但读起来自然。因为每次使用都在具体语境中有功能，极少出现2个以上信号词在同一句中机械叠加。

## 模板信号词表

### 节奏类（单独使用可保留，叠加即模板）
缓缓、轻轻、微微、淡淡、慢慢、渐渐

### 数量/程度类（单独使用可保留，叠加即模板）
一丝、一抹、些许、几分、隐约

### 情态动词类
仿佛、宛如、犹如、不由得、不禁

### 身体反应类（单独可保留，叠加+情态词=模板）
眼中闪过、嘴角勾起、眉头微皱、瞳孔微缩、眉眼低垂

### 心理类（单独可保留，叠加+身体反应词=模板）
心中一动、心头一震、心中暗道、心底泛起

### 呼吸类
深吸一口气、长出口气、倒吸一口凉气

### 书面连接词（叙事中单独使用可，高频即删）
陡然、蓦然、旋即、便见得

## 模板叠加检测

### 检测方法

扫描每句话，统计句中信号词数量：

| 叠加等级 | 一句话中信号词数 | 处理 |
|---------|----------------|------|
| 安全 | 0-1个 | 保留（前提是该词在此语境有功能） |
| 警觉 | 2个 | 检查是否是模板句式，若是→改写 |
| AI模板 | 3个及以上 | 必定是AI模板，必须改写 |

### 典型AI模板句式（出现即改写）

```
信号词叠加模式：
"他缓缓站起身，眼中闪过一丝X，嘴角勾起一抹Y"
→ 3个信号词叠加 + 套话结构 = AI指纹

"她微微一愣，心中涌起一股X，不由得Y"
→ 3个信号词叠加 = AI指纹

"仿佛X一般，带着一丝Y"
→ 2个信号词 + 万能比喻 = AI指纹
```

### 自然使用的范例（信号词单用，不触发改写）

```
"他缓缓转头，看见了血。"
→ 1个信号词，"缓缓"承载动作的凝重感和心理时间

"陈清月叹了口气，坐在屋檐下，满心遗憾和惆怅。"
→ 1个信号词，"叹了口气"在不同章中有不同情感负载

"灶台边，瘦厨子看了眼锅里翻腾的汤汁，张嘴说了一句话。"
→ 无模板信号词，纯动作白描
```

## 语境判断：有功能 vs 无功能

同一个词是否保留，取决于它在当前语境中是否承担功能：

| 词 | 有功能（保留） | 无功能/模板（删除或改写） |
|----|-------------|------------------------|
| 缓缓 | 承载动作节奏、心理时间拉长 | 纯装饰，删掉不影响画面 |
| 一丝 | 强调微量、克制（"嘴角有一丝血迹"） | 与情绪词套话堆叠（"露出一丝微笑"） |
| 仿佛 | 角色视角内的不确定感知（"仿佛是错觉"） | 万能比喻（"仿佛潮水般涌来"） |
| 轻轻 | 动作的真实力度描述 | 每个动作都"轻轻"，变成填充词 |

**判断标准**：删掉这个词后，句子的节奏、画面精度、情绪负载是否有损失？有损失 = 保留，无损失 = 删。

## 替换策略速查

| 原文类型 | 替换方法 | 示例 |
|----------|----------|------|
| 3+信号词叠加句 | 拆成2-3个短句，每句最多1个信号词 | "他缓缓起身，眼中闪过一丝X" → "他站起来。看了她一眼。" |
| 2个信号词模板句式 | 保留有功能的那一个，改写另一个 | "微微皱眉，带着一丝不悦" → "皱了下眉。没说话。" |
| 抽象情绪词 | 具体动作 | "紧张"→"手在抖" |
| "感到XX" | 外化表现 | "感到愤怒"→"攥紧拳头" |
| 形容词堆砌 | 白描手法 | "美丽动人的笑容"→"她笑了" |
| 解释性描写 | 留白 | "他因为害怕而..."→"他退后一步" |
| 连续排比 | 保留最强一条 | 3句排比留1句 |
| 总结升华句 | 直接删除 | "这一刻，她终于明白了..."→删 |

## 全章扫描策略

写完一章后的自检流程：

1. **快速扫读**：朗读正文，标记读起来"滑"的地方——太顺滑没有顿挫 = 可能AI
2. **叠加检测**：grep 信号词，检查是否有同一句中2+信号词叠加
3. **章尾检查**：章尾是否用动作/对话/悬念收束，无总结升华
4. **呼吸感检查**：是否有节奏词（缓缓/轻轻/叹了口气）的适度使用让文本有呼吸？全删光 = 急促干涩
5. **模板句式扫描**：`"眼中闪过.*嘴角"` / `"心中.*不由得"` / `"仿佛.*一般"` 等组合模式

**注意**：不要用"每千字信号词数量"做机械阈值。自然文本（如《每死一次》）中信号词密度可达每千字1-2个，只要不叠加就是安全的。

## 禁用句式模板（保留）

以下句式模板无论信号词多少都应改写：

| 句式 | 示例 | 问题 |
|------|------|------|
| "...，带着..." | "他说，带着一丝无奈" | 万能状语 |
| 陈词滥调/万能比喻 | "像刀子一样锋利" | 公式化比喻；生活化、角色化比喻可保留 |
| "他的声音很轻，却像..." | AI最爱声音描写 | 过于文艺 |
| "仿佛能...一般" | "仿佛能穿透一切一般" | 文言腔 |
| 公式化对话标签 | "好的，"他说道 | 普通"说"可保留；高频公式化时用动作/上下文替代 |
| "他/她感到..." | "她感到一丝失落" | 告诉而非展示 |
| "他/她意识到..." | "他意识到事情不对" | 直接告知 |
| "眼中闪过一丝XX" | "眼中闪过一丝悲伤" | 模板化 |
| "嘴角勾起一抹XX" | "嘴角勾起一抹冷笑" | 模板化 |
| "心中涌起一股XX" | "心中涌起一股暖流" | 模板化 |

## 质量检查清单

- [ ] **无3+信号词叠加句**：每句话最多1个模板信号词
- [ ] **2信号词句已检查**：确认不是模板句式，每个词在该语境有功能
- [ ] **章尾无总结升华**：用动作/对话/悬念收束
- [ ] **无大段心理描写**：心理活动不超过2段
- [ ] **情绪用动作展示**：无直接写"愤怒/伤心/紧张"
- [ ] **对话口语化**：无书面腔，不同角色语气可区分
- [ ] **Show Don't Tell**：用行为代替形容词
- [ ] **无万能比喻**：比喻要么不用，要么生活化/角色化
- [ ] **无三连排比**：AI式的"三个一组"修辞
- [ ] **无论文体**：无"不难看出/由此可见/事实上/综上所述"
