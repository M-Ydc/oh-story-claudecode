# Story Outline Rules

大纲文件的规范。

## Rules

1. **卷纲必填项**：每个卷纲（卷纲_*.md）必须包含以下部分：
   - 本卷目标（主角在本卷结束时达到什么状态）
   - 爽点节奏（每 N 章一个大爽点）
   - 情绪弧线（本卷整体情绪走向）
   - 人物弧线（主要角色在本卷的成长/变化）
   - 伏笔布局（本卷埋设的伏笔及计划回收时间）
   - 关键反转（本卷至少一个关键反转）

2. **细纲必填项**：每个细纲（细纲_*.md）必须包含：
   - 叙事摘要（200-400字，用叙事语气写本章发生了什么。章首钩子在叙事第一句，章尾钩子在叙事最后一句；冲突、信息释放、情绪转折全融入叙事过程）
   - 涉及角色（本章出场主要角色，方便后续加载角色文件）
   - 目标情绪（本章交付什么情绪）
   - 字数目标（X字）

3. **增量写入**：大纲必须增量写入——先写骨架（关键事件和爽点），再逐节填充细节。不要一次性写出所有细节。

4. **与正文同步**：写完一章正文后，回头检查细纲叙事摘要是否与正文一致。如有偏离，更新摘要。

5. **章首钩子**：融入叙事摘要第一句，不单独标注类型字段。技法参考：悬念对话开局 | 闪前碎片 | 倒计时开局 | 神秘独白 | 反差场景 | 未完成动作开局 | 意象预示（详见 `story-setup/references/agent-references/hooks-chapter.md`）

6. **章尾钩子**：融入叙事摘要最后一句，不单独标注类型字段。技法参考：突然揭示 | 紧急危机 | 未完成动作 | 身份反转 | 两难抉择 | 神秘物品/线索 | 倒计时 | 承诺/威胁 | 离奇消失 | 隐藏含义 | 意象钩子 | 回声钩子 | 留白钩子（详见 `story-setup/references/agent-references/hooks-chapter.md`）

7. **每章必须有情绪变化**：叙事摘要读完后应能感受到情绪走向。连续 2 章情绪无变化 = 需调整。

## Examples

### Correct — 完整的卷纲
```
# 卷纲_第一卷.md
## 本卷目标
沈栀从灵力第三层突破至第五层，加入暗卫。
## 爽点节奏
每5章一个大爽点（突破/反杀/身份揭露）
## 情绪弧线
压抑→低谷→觉醒→爆发→余韵
## 人物弧线
沈栀：从隐忍到主动反击
## 伏笔布局
- 伏笔#3：古镜碎片（第8章埋，第二卷回收）
- 伏笔#7：师父真实身份（第12章埋，第三卷回收）
## 关键反转
第18章：暗卫首领竟是沈栀失散多年的兄长
```
所有必填项齐全，伏笔有明确回收计划。

### Correct — 叙事细纲
```
## 细纲：第001章

### 第001章：镜中我

柳州城，陈府厨房。王易蹲在灶台后面等面熟，今天是他十六岁生日。周浩偷加了半勺肉汁，说三小姐修仙回来了。王易笑了笑，许了个愿——希望昨夜美梦成真。

丫鬟柳翠来传话，三小姐要见他们。深夜宅院里少年们被一个个叫进去测灵根。王易看着周浩进了屋子。

一把小刀刺进腹部。周浩的脸在火光里看不清表情。"你是我的劫。"王易倒下去时只看到那双熟悉的手上还沾着面汤的油。

意识沉进一面铜镜。镜中人不是他。王易伸手夺走镜中灵根。镜面翻转，时间倒流。

目标情绪：日常温暖→背叛的震惊与寒意
字数目标：3435字
涉及角色：王易、周浩、柳翠、陈清月
```
叙事摘要用事件本身承载冲突、信息释放和情绪变化。钩子在叙事第一句和最后一句。目标情绪、字数目标、涉及角色各占一行。

### Wrong — 分析式细纲（已淘汰格式，不要使用）
```
## 细纲（第001章）

### 第001章：镜中我
- 核心事件：王易被好友周浩杀死后在镜中夺灵根重生
- 情节点序列：...
- 章首钩子：代入钩 — 厨房日常
- 爽点：无（铺垫章）
- 章尾钩子：悬念钩 — 三小姐传唤
- 场景分解：
  - 场景冲突：...
  - 信息释放：...
  - 情绪转折：...
```
字段式细纲会污染正文语域，导致正文僵硬、AI味浓。已不再使用。
